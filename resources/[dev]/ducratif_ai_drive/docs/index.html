ducratif_ai_drive • Documentation

Conduite autonome IA pour ESX Legacy + ox_lib/ox_target/ox_inventory/oxmysql.

Version 1.0.0 —

Installation

  1. Copiez le dossier dans resources/ et assurez-vous que le nom du dossier est ducratif_ai_drive.
  2. Importez sql/schema.sql et sql/items.sql dans votre base MySQL.
  3. Activez la ressource dans server.cfg.
  4. Modifiez config.lua selon vos besoins.

Ajouter les ITEMS | ox_inventory

  1. Aller dans le dossier ox_inventory/data/items.lua
  2. Importez Le contenu du fichier item.lua dans le dossier docs/item.lua
  3. Pour les images, aller dans ox_inventory/web/images/
  4. Puis modifier le nom des images, par celui de chacun des items:
    chip1.png → renomme en module_ia.png
    chip2.png → renomme en module_ia_premium.png
    chip3.png → renomme en module_ia_platinum.png

Fonctionnement

  • PNJ IA : interaction via ox_target, vérifie la propriété du véhicule à ≤ 25m.
  • Activation IA : achat 1 000 000$, calibrage automatique (3 points générés).
  • Panel N (en voiture) : destination via waypoint du jeu, options, historique, favoris.
  • Arrêt : commande /ia_stop (ou demande police).

Options premium

  • Mode de conduite (souple/rapide/violente)
  • Limitateur de vitesse
  • Régulateur de vitesse
  • Évitement des collisions
  • Slots favoris supplémentaires

Entretien IA

Durabilité configurable : time (minutes) ou distance (km). Items :

  • Module IA : +100% une option
  • Module IA Premium : +50% toutes options
  • Module IA Platine : +100% toutes options

API Natives clés

  • TaskVehicleDriveToCoordLongrange
  • SetEntityMaxSpeed
  • IsVehicleTyreBurst
  • IsVehicleSirenOn
  • HasEntityCollidedWithAnything

Personnalisation

  • PNJ : modèle/coordonnées dans Config.NPC
  • Zones interdites : Config.ForbiddenZones
  • Blacklist : Config.BlacklistClasses/Models
  • Jobs police : Config.PoliceJobs
  • Clés, prix, couleurs UI : Config.*

Logs & Webhook

Chaque action majeure est loggée en BDD ai_logs et envoie un embed si Config.WebhookURL est renseigné.
